# Структура проекта

```
.
├── index.html                 # главная страница GitHub Pages
├── pages/
│   ├── ru/                    # русскоязычные страницы
│   ├── uk/                    # украиноязычные страницы
│   └── dev/                   # тестовые/черновые страницы
├── assets/
│   ├── css/                   # стили
│   ├── js/                    # скрипты
│   └── images/                # изображения и favicon
├── docs/                      # дополнительные текстовые материалы
└── .github/                   # шаблоны GitHub
```

HTML/CSS/JS-код не рефакторился: обновлены только относительные пути к страницам, стилям, скриптам и изображениям после переноса файлов.
